# 视觉资产与即梦 Prompt 规范

这个项目的视觉路线是：即梦生成可循环的场景素材，React 负责交互、分支、按钮、计时器、触发事件和内心独白。

视频和静态图序列都不是交互本身。它们只负责氛围、空间和轻微动作；网页在视觉素材上方叠加对话框、选项、状态提示和专注工具。

## 核心原则

1. 角色先单独设计，不随场景随机生成。
2. 静态图先确认画风、空间、色调和人物位置。
3. 动态视频只加入轻微自然动作；如果先用静态图，就用 2 到 4 张同场景微差图模拟动态。
4. 主角只在需要主角的镜头出现。
5. 不把产品背景、网页交互、按钮、第一幕等说明写进即梦 prompt。
6. 能用画面语言说清楚的内容，不用产品设计语言解释。
7. 中文文字先测试；如果不稳定，后续改成留空并由网页或后期贴图叠加。
8. 每个视频都明确：谁能动，谁不能动，哪些元素必须保持不变。
9. 一个角色，一个身份，一个造型，一个动作，一个镜头语言。
10. 避免“或”“可以”“也可以”等让模型自由选择的表达。

## 静态图序列素材要求

静态图序列适合先替代视频。每个场景给 2 到 4 张同构图、同角色位置、同光线风格的图片，网页会自动轮播，并加轻微缩放漂移。

命名固定为：

```text
01.png
02.png
03.png
04.png
```

当前代码默认读取前三张；需要第四张时我再把对应数组扩展即可。

每组图片的变化应该很小：

- 咖啡杯蒸汽位置变化。
- 窗外人影或车灯位置轻微变化。
- 咖啡师手部动作轻微变化。
- 顾客翻页或打字姿态轻微变化。
- 灯光明暗轻微变化。

不要让构图、人物身份、服装、桌椅位置大幅变化，否则轮播时会像跳切。

## 通用视频素材要求

- 横屏 16:9。
- 每段 5 到 10 秒。
- 镜头稳定，动作慢。
- 适合循环播放。
- 不出现字幕、水印、品牌 logo。
- 不出现不可控的乱码文字。
- 中间或底部画面保持简洁，方便网页叠加 UI。
- 不依赖视频内部人物说话；人物只需要在场和轻微动作。
- 不让用户必须等视频播完；过渡视频控制在 2 到 3 秒。

推荐素材目录：

```text
public/
```

## 主角角色设计 Prompt

```text
2D 半写实视觉小说风格，手绘数字插画质感。设计一位固定主角角色。
角色身份：年轻东亚女性研究生，24 岁左右。黑色中长发，发尾微微自然蓬松，肤色自然，五官清秀真实。她穿米色针织开衫，内搭白色圆领上衣，深灰色直筒长裤，背浅米色帆布包。整体穿着日常、干净、舒适，不时尚夸张，不职业化，不像制服。
角色气质：安静、克制、真实，有一点点疲惫，但仍然愿意重新开始。她不是兴奋状态，也不是悲伤状态，而是刚刚从家里走出来，准备找一个地方坐下来完成一件事。
画面内容：同一个角色的角色设定图。包含六个姿势：正面站立、侧面站立、背面站立、站在门口准备推门、坐下打开电脑、坐着回神思考。所有姿势必须是同一个人、同一套服装、同一个发型、同一个帆布包。
表情内容：同一个角色的四个半身表情：平静、轻微疲惫、专注、轻微放松。表情细腻克制，不夸张。
画面风格：温暖低饱和色调，线条干净，柔和光影，人物比例自然，成人向视觉小说插画风。不要真人实拍，不要 3D 渲染，不要 Q 版，不要夸张动漫眼睛，不要夸张表情，不要赛博朋克，不要奇幻元素。背景保持简单浅色，不要复杂场景。
```

## 咖啡店外景静态图 Prompt

```text
2D 半写实视觉小说风格，手绘数字插画质感。傍晚城市街角的一家温暖咖啡店外景，横屏 16:9。
咖啡店有玻璃门和两扇大窗户，室内透出柔和暖黄色灯光。透过左侧窗户能看到一位顾客坐在窗边阅读。透过右侧窗户能看到一位顾客在桌边使用电脑。吧台区域有一位顾客站着等待咖啡。室内有木质桌椅、暖黄色吊灯、深绿色墙面、暖棕色吧台和一台咖啡机。人物清晰自然，动作安静，不拥挤。
街道刚下过小雨，地面有轻微反光。门口上方有一个简洁中文店名招牌，文字为“片刻咖啡”，字体清晰、干净、自然。吧台后方墙面有一块小菜单板，上面只写四个中文词：拿铁、美式、抹茶、热茶。文字整齐、清楚、不要变形。
构图：咖啡店入口位于画面中间偏右，左侧是湿润街道和窗边区域，底部保持干净，不堆杂物。整体画面温暖、安静、有真实生活感。色调为暖棕色、奶油色、深绿色、琥珀色灯光。线条干净，柔和光影，低饱和。
不要出现主角。不要真人实拍，不要 3D 渲染，不要 Q 版，不要夸张动漫表情，不要赛博朋克，不要奇幻元素，不要品牌 logo，不要英文招牌，不要水印。
```

## 外景视频 Prompt

静态图通过后，再用参考图生成视频。

```text
基于参考图生成 6 到 8 秒横屏视频，16:9，保持原图的 2D 半写实视觉小说风格、咖啡店外观、色调、构图和人物位置一致。
画面只做轻微自然运动：咖啡店内部暖光轻轻变化，窗边读书的顾客轻微翻页，使用电脑的顾客轻轻敲键盘，吧台附近的人轻微移动，街道外面偶尔有一两个行人从远处经过，雨后地面有细微反光变化。
镜头固定，视频结尾保持稳定，整体适合循环播放。主角不要出现，不要有人走到咖啡店门口，不要有人推门进入。
如果原图中有中文招牌和中文菜单，请保持文字稳定清晰，不要让文字变形、闪烁或变成乱码。保持“片刻咖啡”“拿铁”“美式”“抹茶”“热茶”等文字不变。
不要新增人物，不要改变咖啡店结构，不要改变角色服装，不要改变画风，不要变成真人实拍，不要变成 3D。不要水印，不要品牌 logo，不要英文招牌。
```

## 中文文字测试标准

先检查：

1. “片刻咖啡”是否清楚。
2. “拿铁、美式、抹茶、热茶”是否清楚。
3. 文字是否稳定自然，不像乱码。

如果文字效果好，可以保留招牌和菜单板作为场景元素。如果文字效果不好，后续场景让招牌和菜单板留空，由网页或后期贴图叠加中文。
