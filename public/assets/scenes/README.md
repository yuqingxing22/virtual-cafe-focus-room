# Scene Image Sequences

Put still images in these folders to simulate scene motion before full videos are ready.

Use this naming pattern:

```text
01.png
02.png
03.png
```

The app automatically rotates through the images every few seconds. If an image is missing, it falls back to `public/assets/cafe-room.png`.

Recommended folders:

- `entrance/`: cafe exterior before entering.
- `order/`: cafe counter and cafe staff.
- `seat-selection/`: interior view showing seating zones.
- `setup/`: sitting down and preparing the table.
- `focus-window/`: window seat focus room.
- `focus-bar/`: bar seat focus room.
- `focus-corner/`: corner table focus room.
- `focus-quiet/`: quiet zone focus room.
- `complete/`: end-of-session atmosphere.
